hexo 使用技巧整理

hexo new 后同步打开文章编辑器

Hexo 使用 hexo new "post-name" 以后，自动使用编辑器打开文件。

step1 安装 shelljs模块，实现自动部署加载JS脚本

npm install --save-dev shelljs

step2 添加自动脚本

根目录下新建 ‘scripts’ 文件夹，然后新建文件 ‘open-new.js’。文件内容如下：
const exec = require('child_process').exec; // Hexo 3.x hexo.on('new', function (data) { exec(open -a /Applications/Typora.app ${data.path}); });
